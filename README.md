### Command Line Flags
default behavior is 64bit chromedriver <br/>
-f --- Run with 32bit firefox (only works on 32bit machines) <br/>
-h --- Run with 64bit chrome but in headless mode <br/>

#### Example Usage:
./main.py <br/>
./main.py -f <br/>
./main,py -h <br/>
